Kiwi.Plugins.Inventory.InventoryManager Class

Extends Entity
Defined in: src\Inventory-1.0.0.js:14
Module: Plugins
Parent Module: Kiwi

A Kiwi Plugin GameObject that can manage basic inventory items and functionality.

By default, inventory items have an item description, and a count variable which keeps track of how many of the item you currently have. You can add other variables to inventory items with the addVariable method, for example you could add a cost value to each item.
